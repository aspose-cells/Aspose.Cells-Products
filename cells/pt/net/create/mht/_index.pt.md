1.  Acesse a primeira planilha da pasta de trabalho.

{{% blocks/products/pf/agp/code-block title="O código de exemplo mostra como criar o arquivo MHT em C#." offSpacer="" %}}

```cs

// Create Workbook class instance.
Workbook wkb = new Workbook();

// Access the first worksheet of the workbook.
Worksheet sht = wkb.Worksheets[0];

// Get the desired cell(s) of the worksheet.
Cell c00 = sht.Cells["A1"];
Cell c01 = sht.Cells["B1"];
Cell c10 = sht.Cells["A2"];
Cell c11 = sht.Cells["B2"];

// input the value into the cell(s).
c00.PutValue("ColumnA");
c01.PutValue("ColumnB");
c10.PutValue("ValueA");
c11.PutValue("ValueB");

// Save the Workbook as .mht file.
wkb.Save("created_one.mht");

```

Os arquivos MHT, tendo o tipo MIME message/rfc822, encapsulam todo o conteúdo de um arquivo HTML como um único arquivo para armazenamento em arquivamento em dispositivos de armazenamento.
