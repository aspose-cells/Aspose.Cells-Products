{{% blocks/products/pf/agp/text %}}

 Mudah bagi pengembang untuk memasukkan hyperlink ke file excel hanya dalam beberapa baris kode.

{{% /blocks/products/pf/agp/text %}}

1.  Buat instance kelas Buku Kerja.
1.  Tambahkan lembar kerja baru ke objek Excel dan tetapkan nama lembar kerja yang baru ditambahkan.
1.  Akses lembar kerja pertama dari buku kerja.
1.  Tambahkan hyperlink ke URL di sel "A1".
1.  Menambahkan hyperlink internal ke sel "B9" pada lembar kerja lain "Sheet2" di file Excel yang sama.
1.  Gunakan metode Simpan untuk menyimpan buku kerja sebagai file XLSX.
