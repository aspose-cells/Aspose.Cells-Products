 Di excel Anda dapat menambahkan bentuk seperti kotak, lingkaran, dan panah ke dokumen, email, slide, dan spreadsheet. Untuk membebaskan Anda dari banyak penyisipan manual,[Aspose.Cells for Python-Net](https://releases.aspose.com/cells/python-net) API menyediakan fungsionalitas yang sama. API ini sangat mudah digunakan, biasanya hanya diperlukan satu baris kode untuk menyisipkan bentuk tertentu, dan beberapa baris kode dapat menyelesaikan penyisipan sekumpulan bentuk.
