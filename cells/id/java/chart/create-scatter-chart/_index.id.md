{{% blocks/products/pf/agp/content h2="Cara Membuat Grafik Sebar di Java" %}}

{{% blocks/products/pf/agp/text %}}

Perlu membuat Diagram Sebar secara terprogram? Pengembang Java dapat dengan mudah membuat Diagram Sebar hanya dalam beberapa baris kode.

{{% /blocks/products/pf/agp/text %}}

1. Sertakan namespace di file kelas Anda
1.  Membuat[**Buku Kerja**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook/) contoh kelas.
1.  Tambahkan beberapa data ke sel lembar kerja dengan[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/) objek[**putValue**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/#putValue-int-)metode.
1.  Tambah sebuah[**SCATTER**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype/) Bagan ke lembar kerja dengan memanggil[**Grafik**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/) koleksi[**menambahkan**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/#add-int-int-int-int-int-) metode, dikemas dalam[**Lembar kerja**](https://reference.aspose.com/cells/java/com.aspose.cells/worksheet/)obyek.
1.  Akses yang baru[**Bagan**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/)objek dari koleksi Charts dengan meneruskan indeksnya.
1.  Atur sumber data diagram dengan[**Bagan.setChartDataRange**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#setChartDataRange-java.lang.String-boolean-) metode.
1. Simpan sebagai file Excel atau ODS.

{{% blocks/products/pf/agp/code-block title="Contoh kode menunjukkan cara membuat Bagan Sebar di Java." offSpacer="" %}}

```cs

// Instantiating a Workbook object
Workbook workbook = new Workbook();
// Obtaining the reference of the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);

// Adding sample values to cells
Cells cells = worksheet.getCells();
cells.get("A2").putValue("Category1");
cells.get("A3").putValue("Category2");
cells.get("A4").putValue("Category3");

cells.get("B1").putValue("Scatter");
cells.get("B2").putValue(324);
cells.get("B3").putValue(200);
cells.get("B4").putValue(450);

// Adding a Scatter chart to the worksheet
int chartIndex = worksheet.getCharts().add(ChartType.SCATTER, 6, 2, 20, 10);

// Accessing the instance of the newly added chart
Chart chart = worksheet.getCharts().get(chartIndex);

// Setting chart data source as the range "A1:B4"
chart.setChartDataRange("A1:B4", true);

// Save the Workbook as .xlsx file.
workbook.save("output.xlsx");

```
