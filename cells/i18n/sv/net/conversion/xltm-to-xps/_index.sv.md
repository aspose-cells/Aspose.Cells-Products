XLTM-filtillägget representerar filer som genereras av Microsoft Excel som makroaktiverade mallfiler. XLTM-filer liknar XLTX i annan struktur än att de senare inte stöder att skapa mallfiler med makron. Sådana mallfiler används för att generera och ställa in layout, formatering och andra inställningar tillsammans med makron för att underlätta att skapa liknande XLSX-filer då.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="XPS" readMoreLink="https://docs.fileformat.com/page-description-language/xps/" >}}
En XPS-fil representerar sidlayoutfiler som är baserade på XML Paper Specifications skapade av Microsoft. Det utvecklades som en ersättning för EMF-filformat och liknar PDF-filformat, men använder XML i layout, utseende och utskriftsinformation för ett dokument. Det är faktiskt mer berättigat att säga att XPS är ett försök till PDF, men att det inte kunde få tillräckligt med popularitet som ägt av PDF av många anledningar. Microsoft tillhandahåller XPS Document Writer som standard från Windows 7 och framåt för att skapa XPS-filer. XPS-filer kan genereras genom att välja "Microsoft XPS Document Writer" som skrivare medan du skriver ut dokumentet.
