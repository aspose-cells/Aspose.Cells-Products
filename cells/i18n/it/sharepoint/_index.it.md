<p>
 Aspose.Cells per SharePoint è un'app flessibile per la conversione di fogli di lavoro di Excel. Può convertire documenti tra molti formati Microsoft Excel popolari da una raccolta documenti di SharePoint senza dipendenze da Microsoft Excel. Supporta più formati di documento che non sono inclusi nel supporto dei formati di file nativi di SharePoint.
</p>

<div class="container-fluid features-section bg-gray singleproduct">
 <a class="anchor" id="features" name="features">
 </a>
 <div class="row">
  <div class="container">
   <br/>
   <div class="col-lg-12">
    <h2 class="h2title">
     Converti fogli di calcolo Excel in diversi formati
    </h2>
    <p>
     Aspose.Cells per SharePoint consente di convertire fogli di lavoro in molti formati di documenti e immagini popolari dall'interno della raccolta documenti di SharePoint. Aspose.Cells per SharePoint attualmente supporta la maggior parte dei formati di fogli di lavoro insieme a CSV e amp; XML come input e può esportare in formati di fogli di calcolo nonché CSV, HTML, ODS, PDF, XML e amp; formati di immagine.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Concatena i file durante la conversione
    </h2>
    <p>
     Aspose.Cells per SharePoint offre la possibilità di combinare file di formati diversi in un unico documento. Inoltre, puoi anche spostare i documenti nell'elenco in alto o in basso per impostare l'ordine in cui desideri che i documenti vengano combinati.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Converti file in sottocartelle
    </h2>
    <p>
     Aspose.Cells per SharePoint semplifica il processo di conversione convertendo ricorsivamente tutti i documenti e le sottocartelle in una cartella selezionata. La struttura delle sottocartelle esistente viene creata nella cartella di destinazione.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Sovrascrivi i file esistenti
    </h2>
    <p>
     Se la cartella di destinazione contiene già file con lo stesso nome di quelli che desideri convertire, Aspose.Cells per SharePoint ti consente di sovrascrivere i file esistenti oppure puoi semplicemente selezionare un'opzione che indicherà all'app di saltare l'elaborazione di file già esistenti in la cartella di destinazione.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Interagisci con il pubblico API
    </h2>
    <p>
     Per gli sviluppatori che dispongono di soluzioni interne di SharePoint, possono utilizzare Aspose.Cells per il motore di conversione dei documenti di SharePoint tramite il suo API pubblico. Fornisce l'accesso programmatico a tutti i formati supportati e alle opzioni di conversione.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Automazione di Microsoft Excel: non necessaria
    </h2>
    <p>
     Aspose.Cells per SharePoint viene creato utilizzando codice gestito che non necessita mai dell'installazione di Microsoft Excel sulla macchina per funzionare con la conversione del formato Excel. È una perfetta alternativa all'automazione di Microsoft Excel in termini di funzionalità supportate, sicurezza, stabilità, scalabilità, velocità e prezzo.
    </p>
   </div>
  </div>
 </div>
</div>
