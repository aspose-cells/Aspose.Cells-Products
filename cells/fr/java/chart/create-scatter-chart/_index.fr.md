{{% /blocks/products/pf/agp/text %}}

1. Incluez l'espace de noms dans votre fichier de classe
1.  Créer[**Cahier d'exercices**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook/) instance de classe.
1.  Ajoutez des données aux cellules de la feuille de calcul avec le[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/) objets[**putValeur**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/#putValue-int-)méthode.
1.  Ajouter un[**SCATTER**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype/) Graphiquez la feuille de calcul en appelant le[**Graphiques**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/) la collection[**ajouter**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/#add-int-int-int-int-int-) méthode, encapsulée dans le[**Feuille de travail**](https://reference.aspose.com/cells/java/com.aspose.cells/worksheet/)objet.
1.  Accédez au nouveau[**Graphique**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/)objet de la collection Charts en passant son index.
1.  Définissez la source de données du graphique avec[**Chart.setChartDataRange**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#setChartDataRange-java.lang.String-boolean-) méthode.
1. Enregistrez sous forme de fichiers Excel ou ODS.

{{% blocks/products/pf/agp/code-block title="Un exemple de code montre comment créer des graphiques à nuages de points dans Java." offSpacer="" %}}

```cs

// Instantiating a Workbook object
Workbook workbook = new Workbook();
// Obtaining the reference of the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);

// Adding sample values to cells
Cells cells = worksheet.getCells();
cells.get("A2").putValue("Category1");
cells.get("A3").putValue("Category2");
cells.get("A4").putValue("Category3");

cells.get("B1").putValue("Scatter");
cells.get("B2").putValue(324);
cells.get("B3").putValue(200);
cells.get("B4").putValue(450);

// Adding a Scatter chart to the worksheet
int chartIndex = worksheet.getCharts().add(ChartType.SCATTER, 6, 2, 20, 10);

// Accessing the instance of the newly added chart
Chart chart = worksheet.getCharts().get(chartIndex);

// Setting chart data source as the range "A1:B4"
chart.setChartDataRange("A1:B4", true);

// Save the Workbook as .xlsx file.
workbook.save("output.xlsx");

```
