{{% blocks/products/pf/agp/content h2="Como criar gráficos de dispersão em Java" %}}

{{% blocks/products/pf/agp/text %}}

Precisa criar gráficos de dispersão programaticamente? Os desenvolvedores do Java podem criar facilmente gráficos de dispersão em apenas algumas linhas de código.

{{% /blocks/products/pf/agp/text %}}

1. Incluir o namespace em seu arquivo de classe
1.  Criar[**Pasta de trabalho**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook/) instância de classe.
1.  Adicione alguns dados às células da planilha com o[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/) objeto[**colocarValor**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/#putValue-int-)método.
1.  Adicione um[**SCATTER**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype/) Gráfico para a planilha chamando o[**Gráficos**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/) coleção[**adicionar**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/#add-int-int-int-int-int-) método, encapsulado no[**Planilha**](https://reference.aspose.com/cells/java/com.aspose.cells/worksheet/)objeto.
1.  Acesse o novo[**Gráfico**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/)objeto da coleção Charts passando seu índice.
1.  Defina a fonte de dados do gráfico com[**Chart.setChartDataRange**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#setChartDataRange-java.lang.String-boolean-) método.
1. Salve como arquivos Excel ou ODS.

{{% blocks/products/pf/agp/code-block title="O código de exemplo mostra como criar gráficos de dispersão em Java." offSpacer="" %}}

```cs

// Instantiating a Workbook object
Workbook workbook = new Workbook();
// Obtaining the reference of the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);

// Adding sample values to cells
Cells cells = worksheet.getCells();
cells.get("A2").putValue("Category1");
cells.get("A3").putValue("Category2");
cells.get("A4").putValue("Category3");

cells.get("B1").putValue("Scatter");
cells.get("B2").putValue(324);
cells.get("B3").putValue(200);
cells.get("B4").putValue(450);

// Adding a Scatter chart to the worksheet
int chartIndex = worksheet.getCharts().add(ChartType.SCATTER, 6, 2, 20, 10);

// Accessing the instance of the newly added chart
Chart chart = worksheet.getCharts().get(chartIndex);

// Setting chart data source as the range "A1:B4"
chart.setChartDataRange("A1:B4", true);

// Save the Workbook as .xlsx file.
workbook.save("output.xlsx");

```
