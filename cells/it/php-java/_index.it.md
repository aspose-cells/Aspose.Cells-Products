<p>
 Aspose.Cells per PHP tramite Java è una gestione ed elaborazione di fogli di lavoro Excel scalabile e ricca di funzionalità API. Le applicazioni possono eseguire la conversione del formato del foglio di calcolo, la gestione e il rendering dei grafici, l'implementazione di formule, lo stile dei fogli di lavoro al livello più granulare e molto altro. API supporta XLS, XLSX, ODS, SpreadsheetML, CSV, delimitato da tabulazioni, HTML/MHTML, PDF, XPS e altri formati. Gli sviluppatori possono facilmente aggiungere la possibilità di creare, modificare, convertire e renderizzare file Excel nelle loro applicazioni PHP mentre utilizzano JavaBridge.
</p>
