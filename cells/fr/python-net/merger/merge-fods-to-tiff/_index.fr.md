Le format est spécifié dans le cadre des spécifications ODF 1.2 publiées et maintenues par OASIS. Les fichiers FODS ne peuvent pas être ouverts avec Excel, une autre application logicielle de feuille de calcul de Microsoft. Il est capable de décrire des données d'image à deux niveaux, en niveaux de gris, en palette de couleurs et en couleurs dans plusieurs espaces colorimétriques. Il prend en charge les schémas de compression avec perte et sans perte pour choisir entre l'espace et le temps pour les applications utilisant le format.
