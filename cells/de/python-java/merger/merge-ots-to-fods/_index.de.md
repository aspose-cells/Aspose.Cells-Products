Die Anwendungssoftware Calc ähnelt Excel, das in Microsoft Office verfügbar ist. Das OTS-Dateiformat wird zum Erstellen von Vorlagen verwendet, die vordefinierte Einstellungen in Bezug auf Stile, Schriftart, Daten, Tabellenlayout und Formatierung enthalten. OTF-Dateien haben den MIME-Typ application/vnd.oasis.opendocument.spreadsheet-template. Diese Vorlagendateien können als Ausgangspunkt zum Generieren und Speichern tatsächlicher Datendateien verwendet werden, die im Dateiformat ODS gespeichert werden. Das Format ist als Teil der von OASIS veröffentlichten und verwalteten ODF 1.2-Spezifikationen angegeben. FODS-Dateien können nicht mit Excel geöffnet werden, einer anderen Tabellenkalkulationssoftwareanwendung von Microsoft. FODS-Dateien können mit LibreOffice als ODS gespeichert und in andere Formate wie XLS und XLSX konvertiert werden.{{< /blocks/products/pf/agp/i18n/about-file-text >}}
