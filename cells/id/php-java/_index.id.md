<p>
 Aspose.Cells for PHP via Java adalah manajemen dan pemrosesan spreadsheet Excel yang skalabel dan kaya fitur API. Aplikasi dapat melakukan konversi format spreadsheet, manajemen bagan dan rendering, menerapkan formula, gaya lembar kerja ke tingkat yang paling terperinci dan banyak lagi. API mendukung XLS, XLSX, ODS, SpreadsheetML, CSV, Tab Delimited, HTML/MHTML, PDF, XPS dan format lainnya. Pengembang dapat dengan mudah menambahkan kemampuan untuk membuat, mengedit, mengonversi, dan merender file Excel dalam aplikasi PHP mereka saat menggunakan JavaBridge.
</p>
