Das Format ist als Teil der von OASIS veröffentlichten und gepflegten ODF 1.2-Spezifikationen angegeben. Eine Reihe von Anwendungen auf Windows sowie anderen Betriebssystemen können ODS-Dateien zum Bearbeiten und Manipulieren öffnen, darunter Microsoft Excel, NeoOffice und LibreOffice. Das OTS-Dateiformat wird zum Erstellen von Vorlagen verwendet, die vordefinierte Einstellungen in Bezug auf Stile, Schriftart, Daten, Tabellenlayout und Formatierung enthalten. OTF-Dateien haben den MIME-Typ application/vnd.oasis.opendocument.spreadsheet-template.
