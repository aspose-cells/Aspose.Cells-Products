Ce format de fichier a été créé en remplacement du Graphics Interchange Format (GIF) et n'a aucune limitation de droits d'auteur. Cependant, le format de fichier PNG ne prend pas en charge les animations. Le format de fichier PNG prend en charge la compression d’image sans perte, ce qui le rend populaire parmi ses utilisateurs. Au fil du temps, PNG est devenu l’un des formats de fichiers image les plus utilisés.{{< /blocks/products/pf/agp/i18n/about-file-text >}}

Il a été développé en remplacement du format de fichier EMF et est similaire au format de fichier PDF, mais utilise XML dans la mise en page, l'apparence et les informations d'impression d'un document. . Il est, en fait, plus justifié de dire que XPS est une tentative sur PDF, mais n'a pas pu obtenir suffisamment de popularité car il appartient à PDF pour de nombreuses raisons. Microsoft fournit par défaut XPS Document Writer à partir de Windows 7 pour la création de fichiers XPS.
