{{% blocks/products/pf/agp/content h2="Jak utworzyć HTML via Java" %}}

{{% blocks/products/pf/agp/text %}}

 Programiści mogą łatwo tworzyć, ładować, modyfikować i konwertować HTML (Hyper Text Markup Language) w ramach różnych aplikacji raportujących do przetwarzania danych w zaledwie kilku wierszach kodu.

{{% /blocks/products/pf/agp/text %}}

1.  Utwórz instancję[Klasa skoroszytu](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook).
1.  Uzyskaj dostęp do odpowiedniego arkusza za pomocą metody getWorksheets.get().
1.  Wybierz odpowiednią komórkę, wprowadź wartość do żądanej komórki, używając nazwy komórki, np. A1, B3 itd.
1.  Zapisz skoroszyt w formacie HTML przy użyciu metody save().

{{% blocks/products/pf/agp/code-block title="Poniższy kod źródłowy pokazuje, jak utworzyć plik HTML przy użyciu Java." offSpacer="" %}}

```cs

// Create a new workbook
Workbook wkb = new Workbook();

// Access the first worksheet of the workbook.
Worksheet worksheet = wkb.getWorksheets().get(0);

// Add relevant content in the cell
worksheet.getCells().get("A1").putValue("ColumnA");
worksheet.getCells().get("B1").putValue("ColumnB")
worksheet.getCells().get("A2").putValue("ValueA")
worksheet.getCells().get("B2").putValue("ValueB")

// Save the workbook as HTML file
wkb.save("Excel.html"); 

// To enhance the code for further functionalities here are more functions
// getCells() and setValue for modifying the cell content
// getCharts().add() to add charts
// getPivotTables().add() for creating a Pivot Table
// getCells().get(int cell id).setFormula for adding cell level formula

```
