<p>
Aspose.Cells for PHP via Java — это масштабируемое и многофункциональное средство управления и обработки электронных таблиц Excel API. Приложения могут выполнять преобразование формата электронных таблиц, управление и рендеринг диаграмм, реализацию формул, стилизацию рабочих листов на самом детальном уровне и многое другое. API поддерживает XLS, XLSX, ODS, SpreadsheetML, CSV, с разделителями табуляцией, HTML/MHTML, PDF, XPS и другие форматы. Разработчики могут легко добавить возможность создавать, редактировать, конвертировать и отображать файлы Excel в своих приложениях PHP, используя JavaBridge.
</p>
