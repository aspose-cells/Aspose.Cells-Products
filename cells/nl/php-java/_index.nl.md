<p>
 Aspose.Cells voor PHP via Java is een schaalbare en veelzijdige Excel-spreadsheet voor beheer en verwerking API. Toepassingen kunnen conversie van spreadsheetindelingen, diagrambeheer en -weergave uitvoeren, formules implementeren, werkbladen opmaken tot op het meest gedetailleerde niveau en nog veel meer. API ondersteunt XLS, XLSX, ODS, SpreadsheetML, CSV, Tab Delimited, HTML/MHTML, PDF, XPS en andere formaten. Ontwikkelaars kunnen eenvoudig de mogelijkheid toevoegen om Excel-bestanden te maken, bewerken, converteren en renderen in hun PHP-applicaties terwijl ze JavaBridge gebruiken.
</p>
