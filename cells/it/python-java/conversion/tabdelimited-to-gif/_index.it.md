Di proprietà di Unisys, GIF utilizza l'algoritmo di compressione LZW che non degrada la qualità dell'immagine. Per ciascuna immagine GIF in genere consente fino a 8 bit per pixel e sono consentiti fino a 256 colori nell'immagine.
