副檔名為 .mht 的檔案是啟用 MIME 的歸檔檔案格式，它將不同類型的資料包含到單一檔案中。它可以以 CSS 檔案、JavaScript 和其他資源的形式儲存文字、圖像、頁面樣式等資料作為嵌入資源。 MHT 檔案具有 MIME 類型 message/rfc822，將 HTML 檔案的所有內容封裝為單一存檔文件，用於儲存在儲存裝置上存檔。軟體應用程式（例如 Microsoft Word）可讓您透過匯出為 MHT 檔案將 WORD 文件轉換為 MHT。 MHT檔案可以使用流行的瀏覽器打開，例如Microsoft Internet Explore和Google Chrome。
