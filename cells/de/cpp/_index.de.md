<p>
 Aspose.Cells für C++ ist ein natives C++ API für die Bearbeitung von Tabellenkalkulationen. Anwendungen können Funktionen wie das Erstellen, Konvertieren und Bearbeiten der integrierten und benutzerdefinierten Eigenschaften von Tabellenkalkulationen, Themen, Stile und Formatierungen, Datenvalidierung, bedingte Formatierung, Lesen, Schreiben und Berechnen von Formeln, Gruppieren von Zeilen und Spalten und vieles mehr ausführen, ohne auf Office-Automatisierung angewiesen zu sein oder Microsoft-Excel. API unterstützt die am häufigsten verwendeten Tabellenkalkulationsformate wie XLS, XLSX, XLSM und XLSB.
</p>
