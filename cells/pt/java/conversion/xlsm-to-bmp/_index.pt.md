{{% blocks/products/pf/agp/code-block title="Código-fonte de conversão de XLSM para BMP Java" offSpacer="" %}}

```cs
// carregue o arquivo XLSM a ser renderizado
Workbook workbook = new Workbook("sourceFile.xlsm");
// acessar a planilha padrão da coleção
Worksheet worksheet = workbook.getWorksheets().get(0);
// definir parâmetros para a imagem resultante
ImageOrPrintOptions options = new ImageOrPrintOptions();
options.setOnePagePerSheet(true);
options.setImageType(ImageType.BMP);
// converter planilha para imagem no formato BMP
SheetRender renderer = new SheetRender(worksheet, options);
renderer.toImage(0, "output.bmp");   


```

Uma macro é usada para registrar as etapas que são executadas repetidamente e facilita a execução das ações executando a macro novamente. Arquivos com extensão .BMP representam arquivos de imagem de bitmap que são usados para armazenar imagens digitais de bitmap. Essas imagens são independentes do adaptador gráfico e também são chamadas de formato de arquivo de bitmap independente de dispositivo (DIB). Essa independência serve para abrir o arquivo em várias plataformas, como Microsoft Windows e Mac. O formato de arquivo BMP pode armazenar dados como imagens digitais bidimensionais em formato monocromático e colorido com várias profundidades de cor.
