I file OTF hanno application/vnd.oasis.opendocument.spreadsheet-template di tipo MIME.         {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="GIF" readMoreLink="https://docs.fileformat.com/image/gif/" >}}Un formato GIF o Graphical Interchange Format è un tipo di immagine altamente compressa. Di proprietà di Unisys, GIF utilizza l'algoritmo di compressione LZW che non degrada la qualità dell'immagine. Per ciascuna immagine GIF in genere consente fino a 8 bit per pixel e sono consentiti fino a 256 colori nell'immagine. Un GIF animato combina numerose immagini o fotogrammi in un unico file e li visualizza in sequenza per generare una clip animata o un breve video.
