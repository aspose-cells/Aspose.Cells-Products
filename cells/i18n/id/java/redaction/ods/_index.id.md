```cs
Workbook workbook = new Workbook(dataDir + "sourceFile.ods");

Worksheet worksheet = workbook.getWorksheets().get(0);

// Tentukan rentang di mana Anda ingin mencari
// Di sini kisarannya adalah E3:H6
CellArea area = CellArea.createCellArea("E3", "H6");

// Tentukan opsi Temukan
FindOptions opts = new FindOptions();
opts.setLookInType(LookInType.VALUES);
opts.setLookAtType(LookAtType.ENTIRE_CONTENT);
opts.setRange(area);

Cell cell = null;

do {
	// Cari sel dengan pencarian nilai dalam jangkauan
	cell = worksheet.getCells().find("search", cell, opts);

	// Jika tidak ada sel seperti itu yang ditemukan, maka putuskan loop
	if (cell == null)
		break;

	// Ganti sel dengan nilai ganti
	cell.putValue("replace");

} while (true);

// Simpan buku kerja
workbook.save(dataDir + "output.ods");


```

Ini adalah format berbasis XML dan merupakan salah satu dari beberapa subtipe dalam keluarga Open Document Formats (ODF).
