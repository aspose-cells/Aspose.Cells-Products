<p>
 Aspose.Cells pour SharePoint est une application de conversion de feuille de calcul Excel flexible. Il peut convertir des documents entre de nombreux formats Microsoft Excel populaires à partir d'une bibliothèque de documents SharePoint sans dépendances Microsoft Excel. Il prend en charge plusieurs formats de document qui ne sont pas inclus dans la prise en charge du format de fichier natif de SharePoint.
</p>

<div class="container-fluid features-section bg-gray singleproduct">
 <a class="anchor" id="features" name="features">
 </a>
 <div class="row">
  <div class="container">
   <br/>
   <div class="col-lg-12">
    <h2 class="h2title">
     Convertir des feuilles de calcul Excel en plusieurs formats
    </h2>
    <p>
     Aspose.Cells pour SharePoint permet de convertir des feuilles de calcul en de nombreux formats de documents et d'images populaires à partir de la bibliothèque de documents SharePoint. Aspose.Cells pour SharePoint prend actuellement en charge la plupart des formats de feuille de calcul, ainsi que CSV &amp ; XML comme entrée, et peut exporter vers des formats de feuille de calcul ainsi que CSV, HTML, ODS, PDF, XML &amp; formats d'images.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Concaténer des fichiers lors de la conversion
    </h2>
    <p>
     Aspose.Cells pour SharePoint offre la possibilité de combiner des fichiers de différents formats en un seul document. De plus, vous pouvez également déplacer des documents dans la liste vers le haut ou vers le bas pour définir l'ordre dans lequel vous souhaitez que les documents soient combinés.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Convertir des fichiers dans des sous-dossiers
    </h2>
    <p>
     Aspose.Cells pour SharePoint facilite le processus de conversion en convertissant de manière récursive tous les documents et sous-dossiers d'un dossier sélectionné. La structure de sous-dossiers existante est créée dans le dossier de destination.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Écraser les fichiers existants
    </h2>
    <p>
     Si le dossier de destination contient déjà des fichiers portant les mêmes noms que ceux que vous souhaitez convertir, Aspose.Cells pour SharePoint vous permet d'écraser les fichiers existants ou vous pouvez simplement sélectionner une option qui demandera à l'application d'ignorer le traitement des fichiers qui existent déjà dans le dossier de destination.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Interagissez avec le public API
    </h2>
    <p>
     Pour les développeurs disposant de solutions SharePoint internes, ils peuvent utiliser Aspose.Cells pour le moteur de conversion de documents de SharePoint via son API public. Il fournit un accès programmatique à tous les formats pris en charge et aux options de conversion.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Automatisation Microsoft Excel - Non nécessaire
    </h2>
    <p>
     Aspose.Cells pour SharePoint est créé à l'aide d'un code géré qui n'a jamais besoin d'installer Microsoft Excel sur la machine pour fonctionner avec la conversion de format Excel. C'est une alternative parfaite à l'automatisation de Microsoft Excel en termes de fonctionnalités prises en charge, de sécurité, de stabilité, d'évolutivité, de vitesse et de prix.
    </p>
   </div>
  </div>
 </div>
</div>
