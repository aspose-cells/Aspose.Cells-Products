<p>
 Aspose.Cells for Node.js via Java — это многофункциональное и масштабируемое приложение API для обработки электронных таблиц Excel с использованием Node.js. API предлагает преобразование формата файлов Excel, стилизацию рабочих листов на самом детальном уровне, создание диаграмм и т.д. рендеринг, надежный механизм расчета формул и многое другое.
</p>

<p>
 Aspose.Cells for Node.js via Java поддерживает большинство популярных форматов электронных таблиц, включая XLS, XLSX, ODS, SpreadsheetML, CSV, TSC &amp; HTML.
</p>
