```cs
Workbook workbook = new Workbook(dataDir + "sourceFile.xlsb");

Worksheet worksheet = workbook.getWorksheets().get(0);

// Aramak istediğiniz aralığı belirtin
// Burada aralık E3:H6'dır.
CellArea area = CellArea.createCellArea("E3", "H6");

// Bul seçeneklerini belirtin
FindOptions opts = new FindOptions();
opts.setLookInType(LookInType.VALUES);
opts.setLookAtType(LookAtType.ENTIRE_CONTENT);
opts.setRange(area);

Cell cell = null;

do {
	// Aralık içinde değer arama ile hücreyi arayın
	cell = worksheet.getCells().find("search", cell, opts);

	// Böyle bir hücre bulunamadıysa, döngüyü kırın
	if (cell == null)
		break;

	// Hücreyi değer değiştirme ile değiştirin
	cell.putValue("replace");

} while (true);

// Çalışma kitabını kaydet
workbook.save(dataDir + "output.xlsb");


```

İçerik, yapılandırılmamış veya yarı yapılandırılmış sayı tabloları, metin veya hem sayılar hem de metin, formüller, harici veri bağlantıları, çizelgeler ve resimler içerebilir. XLSX (ve daha önce XLS), çalışma kitaplarını kaydetmek için en yaygın kullanıcı tarafından seçilen dosya biçimleri olduğundan, XLSB nadiren çalışma kitaplarını depolamak için kullanılır.
