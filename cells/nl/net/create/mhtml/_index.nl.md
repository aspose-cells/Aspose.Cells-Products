```cs

// Werkmapklasse-instantie maken.
Workbook wkb = new Workbook();

// Open het eerste werkblad van de werkmap.
Worksheet sht = wkb.Worksheets[0];

// Verkrijg de gewenste cel(len) van het werkblad.
Cell c00 = sht.Cells["A1"];
Cell c01 = sht.Cells["B1"];
Cell c10 = sht.Cells["A2"];
Cell c11 = sht.Cells["B2"];

// voer de waarde in de cel(len) in.
c00.PutValue("ColumnA");
c01.PutValue("ColumnB");
c10.PutValue("ValueA");
c11.PutValue("ValueB");

// Sla de werkmap op als .mhtml-bestand.
wkb.Save("created_one.mhtml");


```

Bestanden met de extensie MHTML vertegenwoordigen een archiefindeling voor webpagina's die door een aantal verschillende toepassingen kan worden gemaakt. Het formaat staat bekend als archiefformaat omdat het de web-HTML-code en bijbehorende bronnen in één bestand opslaat. Deze bronnen omvatten alles dat aan de webpagina is gekoppeld, zoals afbeeldingen, applets, animaties, audiobestanden enzovoort. MHTML-bestanden kunnen in verschillende toepassingen worden geopend, zoals Internet Explorer en Microsoft Word. Microsoft Windows gebruikt de MHTML-bestandsindeling voor het opnemen van scenario's van problemen die zijn waargenomen tijdens het gebruik van een toepassing op Windows die problemen oplevert. Het MHTML-bestandsformaat codeert de pagina-inhoud vergelijkbaar met de specificaties die zijn gedefinieerd in message/rfc822, wat e-mailgerelateerde specificaties in platte tekst zijn.
