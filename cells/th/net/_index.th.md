Aspose.Cells for .NET เป็นการเขียนโปรแกรมสเปรดชีต Excel API เพื่อเพิ่มความเร็วในการจัดการสเปรดชีตและงานประมวลผล API รองรับการสร้างแอปพลิเคชันข้ามแพลตฟอร์มที่มีความสามารถในการสร้าง แก้ไข แปลง เรนเดอร์ และพิมพ์สเปรดชีต ยิ่งไปกว่านั้น มันไม่ได้ขึ้นอยู่กับ Microsoft Excel หรือ Microsoft คอมโพเนนต์ Office Interop ที่จะติดตั้ง แต่เป็นชุด API ที่มีประสิทธิภาพซึ่งจัดการกับรูปแบบ Excel ทั้งหมด เช่นเดียวกับรูปแบบ CSV และ SpreadsheetML

</p>
