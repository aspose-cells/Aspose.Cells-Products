        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="XLT" readMoreLink="https://docs.fileformat.com/spreadsheet/xlt/" >}}
 Les fichiers portant l'extension .XLT sont des fichiers modèles créés avec Microsoft Excel, un tableur faisant partie de la suite Office Microsoft. Microsoft Office 97-2003 prenait en charge la création de nouveaux fichiers XLT ainsi que leur ouverture. La dernière version d'Excel est toujours capable d'ouvrir les fichiers modèles dans cet ancien format. Un tel fichier modèle est utilisé pour créer rapidement de nouveaux fichiers Excel avec des données et des paramètres par défaut tels que le formatage de la page, la taille de la police, les marges, les graphiques, etc. qui peuvent être ensuite enregistrés en tant que nouveaux fichiers .XLS.
