<p>
 Aspose.Cells pour PHP via Java est un tableur Excel évolutif et riche en fonctionnalités de gestion et de traitement API. Les applications peuvent effectuer la conversion du format de feuille de calcul, la gestion et le rendu des graphiques, la mise en œuvre de formules, le style des feuilles de calcul au niveau le plus granulaire et bien plus encore. API prend en charge XLS, XLSX, ODS, SpreadsheetML, CSV, Tab Delimited, HTML/MHTML, PDF, XPS et d'autres formats. Les développeurs peuvent facilement ajouter la possibilité de créer, modifier, convertir et afficher des fichiers Excel dans leurs applications PHP tout en utilisant JavaBridge.
</p>
