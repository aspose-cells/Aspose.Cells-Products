{{% blocks/products/pf/agp/content h2="Как создать точечные диаграммы в Java" %}}

{{% blocks/products/pf/agp/text %}}

Нужно создавать точечные диаграммы программно? Разработчики Java могут легко создавать точечные диаграммы всего за несколько строк кода.

{{% /blocks/products/pf/agp/text %}}

1. Включите пространство имен в файл класса
1.  Создавать[**Рабочая тетрадь**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook/) экземпляр класса.
1.  Добавьте данные в ячейки листа с помощью[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/) объекты[**putValue**](https://reference.aspose.com/cells/java/com.aspose.cells/cell/#putValue-int-)метод.
1.  Добавить[**SCATTER**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype/) Создайте диаграмму на рабочем листе, вызвав[**Графики**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/) коллекция[**добавлять**](https://reference.aspose.com/cells/java/com.aspose.cells/chartcollection/#add-int-int-int-int-int-) метод, инкапсулированный в[**Рабочий лист**](https://reference.aspose.com/cells/java/com.aspose.cells/worksheet/)объект.
1.  Доступ к новому[**Диаграмма**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/)объект из коллекции Charts, передав его индекс.
1.  Установите источник данных диаграммы с помощью[**Chart.setChartDataRange**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#setChartDataRange-java.lang.String-boolean-) метод.
1. Сохраните в формате Excel или ODS.

{{% blocks/products/pf/agp/code-block title="Пример кода показывает, как создавать точечные диаграммы в Java." offSpacer="" %}}

```cs

// Instantiating a Workbook object
Workbook workbook = new Workbook();
// Obtaining the reference of the first worksheet
Worksheet worksheet = workbook.getWorksheets().get(0);

// Adding sample values to cells
Cells cells = worksheet.getCells();
cells.get("A2").putValue("Category1");
cells.get("A3").putValue("Category2");
cells.get("A4").putValue("Category3");

cells.get("B1").putValue("Scatter");
cells.get("B2").putValue(324);
cells.get("B3").putValue(200);
cells.get("B4").putValue(450);

// Adding a Scatter chart to the worksheet
int chartIndex = worksheet.getCharts().add(ChartType.SCATTER, 6, 2, 20, 10);

// Accessing the instance of the newly added chart
Chart chart = worksheet.getCharts().get(chartIndex);

// Setting chart data source as the range "A1:B4"
chart.setChartDataRange("A1:B4", true);

// Save the Workbook as .xlsx file.
workbook.save("output.xlsx");

```
